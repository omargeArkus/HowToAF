# ActiveFit+ Beta Onboarding Guide

A lightweight, single-page tutorial for beta testers to install and activate the ActiveFit+ beta build via TestFlight.

## What's Inside

- `index.html` — the tutorial page
- `Videos/` — walkthrough videos for each phase
- `images/` — logo and images

## How to View

Open the link shared with you by the ActiveFit+ team. No account or installation required — just open it in any browser on your phone or computer.

## Tutorial Phases

| Phase | Description |
|-------|-------------|
| 1 | Installing TestFlight |
| 2 | Installing ActiveFit+ |
| 3 | Activating the Beta Request |
| 4 | Installing the Beta Build |

## For Maintainers

To add or replace a video, drop the `.mp4` file inside the `Videos/` folder and update the `<source src="...">` path in `IndexNP.html` for the corresponding phase.

To update the tutorial text, edit the step content directly inside `IndexNP.html` — each phase is clearly commented.
